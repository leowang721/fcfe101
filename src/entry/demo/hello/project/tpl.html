Initialize Context

  1. 安装 nodejs：brew install nodejs
  2. 安装 edp：npm i -g edp edp-build
  3. 安装 edpx-fc：edp fc
  4. 看看 edp：edp
  5. 看看 efc：efc --help

Coding Steps

  1. 创建项目目录：mkdir project && cd project
  2. 项目初始化：edp project init .
  3. 适当引入需要的库：edp import fc-view fc-ajax fc-storage jquery
  4. 增加 html：efc add main.html
  5. 增加 js：efc add src/main.js
  6. 更多：efc --help
